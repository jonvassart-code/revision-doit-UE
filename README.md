# 🇪🇺 Fiches de Révision — Droit de l'Union Européenne

Interface interactive de révision pour le cours de **Droit de l'Union Européenne** (UCLouvain, 2025-2026, Prof. F. Dopagne).

## 📋 Contenu

L'application comporte **3 onglets** :

### 📝 Fiches de révision
- 11 chapitres complets (Intro + Ch. I à X)
- ~85 points clés dépliables
- Barre de recherche instantanée
- Navigation par chapitre avec code couleur

### 🎤 Questions d'oral
- 50 questions ouvertes type examen oral
- Réponses masquées avec bouton « Voir la réponse »
- Filtrage par chapitre
- Bouton « Révéler tout / Masquer tout »

### 🧠 Par cœur
- Tout ce qui n'est PAS dans le code juridique
- Arrêts fondateurs (noms, dates, apports)
- Formules jurisprudentielles
- Chronologie historique
- Chiffres et seuils clés
- Concepts doctrinaux et distinctions

## 🚀 Utilisation

Ce fichier est un composant **React (.jsx)** conçu pour fonctionner dans :
- [Claude.ai](https://claude.ai) (artifacts React)
- Tout projet React/Vite/Next.js

### Dans un projet React existant

```bash
npm create vite@latest revision-app -- --template react
cd revision-app
cp revision_droit_ue.jsx src/App.jsx
npm install
npm run dev
```
